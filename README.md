# Nike Shoes App 👟

A modern and clean Flutter UI for a shoes shopping application inspired by Nike design.

## ✨ Features
- Modern UI design
- Discount banner
- Category selector (All, Running, Sneakers...)
- Product cards with price
- GridView layout
- Scrollable interface
